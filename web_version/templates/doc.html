Introduction

  • Item Name : Sleep Cycle Visualization (SCVIS) tool
  • Item Version : v 1.0
  • Author : Leping Li and team

SCVIS is designed to extract, customize and visualize sleep cycles from EEG data. This documentation is to walk users through the both the web and desktop versions of the tool.

Desktop version

The desktop version of SCVIS can be downloaded here. In order to use the desktop version, you will need to install the following tools and packages.

  1. 1. Install python 3.6 or higher
  2. 2. Install the following python packages: numpy, matplotlib, mne

Inputs and Outputs #back to top

Inputs

SCVIS takes two types of input files.

  1. 1. Sleep data in EDF format
  2. 2. Epoch wise sleep stages in text format

Outputs

SCVIS generates following outputs.

  1. 1. An image file for the visualization in JPEG format
  2. 2. Epoch wise sleep cycles in text format
  3. 3. Epoch wise sleep stages in text format
